Spherical Airship

This form can be used to create spherical airship configuration files or mission requirements files for use with the KBE tool, developed to analyze and optimize spherical airships. The configuration file format produced using this utility is .xlsx.

Select your mode of operation below. Forward mode defines the aircraft's configuration for performance analysis. Inverse mode defines a required mission profile to generate an optimized configuration.
